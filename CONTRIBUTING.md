# Contributing

Thank you for your interest in contributing to the Hostking Password Generator!

## How to contribute

1. Fork the repository
2. Create a branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Test thoroughly in multiple browsers
5. Commit your changes: `git commit -m 'Add: your feature description'`
6. Push to the branch: `git push origin feature/your-feature-name`
7. Open a pull request

## Code standards

- Vanilla JavaScript only (no frameworks or libraries)
- No external CDN dependencies
- All changes must work offline
- Code must not send any data to external servers
- Follow the existing code style

## Security

If you discover a security issue, please report it via the [SECURITY.md](SECURITY.md) process rather than opening a public issue.

## Feature ideas

- Passphrase generator
- Bulk password generation
- Password history (session-only, never persisted)
- QR code export
- Entropy display
- Estimated crack time
- Custom character exclusion list
- Pronounceable password mode

## Questions

Open a GitHub Issue for any questions or discussions.
