# 🔐 Password Generator

A lightweight, open-source password generator built with HTML, CSS and JavaScript.

Generate secure passwords directly in your browser with no data sent to any server.

## Features

- Adjustable password length (8–64 characters)
- Uppercase letters
- Lowercase letters
- Numbers
- Symbols
- One-click copy to clipboard
- Fully client-side
- Responsive design
- No tracking or analytics

## Usage

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Choose your desired options.
4. Generate a secure password.

## Why use a password generator?

Strong passwords help protect:

- Web hosting accounts
- Email accounts
- WordPress websites
- VPS servers
- SSH logins
- Databases
- Control panels
